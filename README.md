Images of app later

# What does it do?
* Roll dices from selection of:
  * K4
  * K6
  * K8
  * K10
  * K12
  * K20
* Generate numbers from range that is specified by user.
* Count HitPoints/Lives:
  * Number of players from 2 - 6
  * Number of HP from 25 to 40 (increase of HP by 5)
